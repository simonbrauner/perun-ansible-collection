## Notes for Large OpenAPI documents
If the OpenAPI document is large, imports in perun_openapi.apis and perun_openapi.models may fail with a
RecursionError indicating the maximum recursion limit has been exceeded. In that case, there are a couple of solutions:

Solution 1:
Use specific imports for apis and models like:
- `from perun_openapi.api.default_api import DefaultApi`
- `from perun_openapi.model.pet import Pet`

Solution 2:
Before importing the package, adjust the maximum recursion limit as shown below:
```
import sys
sys.setrecursionlimit(1500)
import perun_openapi
from perun_openapi.apis import *
from perun_openapi.models import *
```
